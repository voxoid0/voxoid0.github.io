security error

Error

Agreeing to the Xcode/iOS license requires admin privileges, please run "sudo xcodebuild -license" and then retry this command.

If you're in a hurry like I was, you may see 'admin privileges' and 'sudo', and immediately try "sudo ./gradlew ...", which may work but will re-download all of your dependencies into root's gradle dependencies folder (/var/.m2), including gradle itself. It didn't work for me; this could have been because I also have a manually-installed dependency that doesn't exist in any of the maven or gradle repositories online.

You're getting this message because you haven't agreed to Apple's terms regarding your certificates/codesign, so just do that real quick, and then gradle should be able to invoke codesign for you after that. Run sudo xcodebuild -license like it says, scroll through pages upon pages of stuff you probably won't read, and enter agree at the end.

./gradlew createApp codeSign createDmg

codesign -s 'Developer ID Application: Joel Becker' app/build/distributions/Percebe\ 1.2.5.dmg
